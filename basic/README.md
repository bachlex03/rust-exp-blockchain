# Repository for Rust experience

## CLI's

- check rust version: `rustc --version`
- check cargo version: `cargo --version`
- Init new Rust project: `cargo new "project-name"`
- Run project: `cargo run`
- Run project but details cli: `cargo run --package basic --bin basic`
- build project: `cargo build`
- clean a builded project: `cargo clean`

Cargo: the Rust package manager and build system. Cargo handles project creation, dependency management, building, testing, and running Rust projects.
